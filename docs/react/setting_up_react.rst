=================
Setting up React
=================

Install npm
--------------

1. Go to the website
 * https://nodejs.org/en/

2. Download npm

Install depedencies
--------------------
1. run:

    ::

        npm install

Have npm watch for changes
---------------------------

1. run:

    ::

        npm run dev

To Build the Production files
--------------------------------
* This creates minified version of files

1. run:

    ::

        npm run build
